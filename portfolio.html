Lattice Structures for Hypersonic Spacecraft

Over the last year, I participated in hypersonic research, working on lattice structures for spacecraft returning into the atmosphere. Using Inventor, I was able to develop 3 strong, complex designs that were later tested in other simulation software to validate the structure and heat resistance.

Project Moonstone Tasks

Onboarding Project Moonstone, I was able to develop 3D models to train AI on how to compute the designs through prompting, constructive rubrics, and visible model references. Having a CAD proficient approach to this project taught me how to develop strong input prompts so that an AI model could produce the CAD.
